# Boxing & MMA Combat Sports Website

Welcome to the Boxing & MMA Combat Sports Website repository! This project is a comprehensive platform for boxing and MMA enthusiasts, providing the latest fight events, fighter profiles, training resources, and community forums.

## Technologies Used

- **React.js**: For building the user interface and managing the component state.
- **Vite**: A fast build tool that provides an optimized development experience.
- **Firebase**: For authentication, real-time database, and hosting.
- **EmailJS**: For integrating email functionalities without a backend server.
- **Node.js**: Used for server-side operations and API integrations.
- **Tailwind CSS**: For utility-first CSS styling and responsive design.

## Features

- **Event Coverage**: Stay updated with the latest boxing and MMA events.
- **Fighter Profiles**: Detailed profiles of fighters with stats and biographies.
- **Training Resources**: Access to training guides and tips.
- **Community Forums**: Engage with other fans and fighters in discussions.
